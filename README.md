# Markdown Table Editor

ExcelのようなインターフェースでMarkdownの表を編集できるVS Code拡張機能です。

## 機能
- **Markdown表の編集**: `Ctrl+Shift+E` (Mac: `Cmd+Shift+E`) でカーソル位置の表をWebviewで開きます。※コマンドパレットから `Edit Markdown Table` を実行することも可能です。
- **Excel風UI**: 直感的に行や列を追加・削除・編集できます。
- **ライブ同期**: 編集内容はリアルタイムでMarkdownファイルに反映されます。

## ショートカットキー (Webview内)
- **行追加 (下)**: `Ctrl` + `Enter` （またはツールバーの「Add Row」ボタン）
- **列追加 (右)**: ツールバーの「Add Column」ボタン
- **移動**: `Enter` キーで右のセルへ移動（行末の場合は次の行の先頭へ）
- **行削除**: 行の左側にある「×」ボタンをクリック

## インストール
1. このリポジトリをクローンまたはダウンロードします。
2. VS Codeで開きます。
3. `npm install` を実行して依存関係をインストールします。
4. `F5` を押してデバッグ実行を開始します。
